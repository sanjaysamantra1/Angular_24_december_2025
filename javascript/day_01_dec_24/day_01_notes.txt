Web Development
===============
-Developing a website

1. FrontEnd : User Interface ( Look & Feel )
        HTML : Content ( paragraph, image,form,audio,video)
        CSS :  Styling ( border, backgorund , width , height, padding, margin )
        Bootstrap : CSS Library (css classnames)
        Javascript : Behaviour
        Angular/React : Javascript framework/Library (RAD,CLI,Form,API,Routing) 

2. BackEnd : Develop APIs ( java , .net, Python, PHP , Nodejs, go )

3. DataBase : Holds the data ( oracle , mysql , mongodb )

CSS ( cascading style sheet)
====
1. inline css (style attribute) ( Restricted to 1 element )
   <h2 style="color: blue">My First HTML Program</h2>

2. internal css ( style tag ) ( Restricted to 1 HTML Page )
   <style></style>

3. external css ( project level )
   a. write css code in a separate .css file
   b. include that css file in HTML file using <link>


How to Use bootstrap in HTML Files
==================================
-CSS library : pre-defined css classnames
-Maintained by Twitter
-Bootstrap 5 is the latest version
-https://getbootstrap.com/docs/5.3/getting-started/introduction/
